# Phase 3 Remediation Report

## Objective

Prove that every statistical procedure receives correctly oriented matrices and aligned observations.

## Remediation Summary

- Established one canonical matrix contract: rows are aligned observations, columns are models.
- Added shared score-matrix validation for statistical consumers.
- Added paired-observation validation for Wilcoxon and paired effect-size paths.
- Rejected duplicate `(repetition_id, fold_id)` keys during score-matrix construction.
- Changed stored significance effect sizes to use paired differences.
- Replaced the matrix-orientation audit with a complete source-to-consumer trace.
- Expanded synthetic benchmark tests for orientation, alignment, ranks, and paired effects.

## Scientific Findings

- Friedman receives column vectors from a matrix whose rows are common observations.
- Pairwise Wilcoxon receives equal-length vectors sliced from identical observation rows.
- Nemenyi ranks are generated row by row from the same observations used by Friedman.
- Confidence intervals are computed per model column.
- Effect sizes generated by the storage significance pipeline now use paired differences from aligned observation rows.
- No transpose or reshape operations are present in the audited statistical path.
