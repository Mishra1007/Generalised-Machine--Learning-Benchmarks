# Datasets

This directory handles all dataset operations.

## Purpose

- Load datasets from various sources (sklearn, local files, remote URLs)
- Cache downloaded datasets locally
- Manage dataset versioning and metadata
- Handle data validation and integrity checks

## Structure

```
datasets/
├── loaders.py          # Dataset loading utilities
├── registry.py         # Dataset registry and metadata
├── cache/              # Local dataset cache
└── metadata.json       # Dataset specifications
```

## Supported Sources

- Scikit-learn built-in datasets (iris, breast_cancer, etc.)
- Local CSV/Parquet files
- Remote URLs
- Custom data generators

## Usage

```python
from datasets.loaders import load_dataset

X, y = load_dataset("iris")
```
