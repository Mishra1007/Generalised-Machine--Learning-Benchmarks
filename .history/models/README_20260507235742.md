# Models

This directory manages model definitions, configurations, and registry.

## Purpose

- Define and wrap scikit-learn models with consistent interfaces
- Maintain a registry of available models
- Store model configurations and hyperparameters
- Enable easy model switching and comparison

## Structure

```
models/
├── registry.py         # Model registry and factory
├── wrappers.py         # Model wrappers for consistency
├── configs.py          # Default hyperparameters
└── __init__.py         # Public API
```

## Model Registry

Central registry allows:
- Dynamic model instantiation
- Consistent hyperparameter management
- Easy model discovery and comparison

## Supported Models (Scikit-learn)

- Logistic Regression
- Random Forest
- Gradient Boosting
- SVM
- Naive Bayes
- KNN
- Decision Trees
- Linear/Ridge/Lasso Regression

## Usage

```python
from models import get_model

model = get_model('random_forest', n_estimators=100)
```
