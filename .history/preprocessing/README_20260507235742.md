# Preprocessing

This directory contains all data preprocessing and feature engineering logic.

## Purpose

- Feature scaling and normalization
- Feature engineering and transformation
- Handling missing values
- Categorical encoding
- Feature selection
- Train-test splitting with stratification

## Structure

```
preprocessing/
├── scalers.py          # Feature scaling pipelines
├── transformers.py     # Feature transformations
├── selectors.py        # Feature selection methods
├── pipelines.py        # Scikit-learn Pipeline compositions
└── validators.py       # Data validation utilities
```

## Philosophy

- Use scikit-learn's Pipeline for reproducibility
- Fit on training data only (prevent data leakage)
- Document all transformations for analysis

## Example

```python
from sklearn.pipeline import Pipeline
from sklearn.preprocessing import StandardScaler
from sklearn.feature_selection import SelectKBest

pipeline = Pipeline([
    ('scaler', StandardScaler()),
    ('selector', SelectKBest(k=10))
])
```
