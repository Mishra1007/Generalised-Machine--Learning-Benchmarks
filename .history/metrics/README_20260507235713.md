# Metrics

This directory houses evaluation metrics and scoring utilities.

## Purpose

- Define standard evaluation metrics
- Compute performance scores consistently
- Generate metric summaries and aggregations
- Support both classification and regression tasks

## Structure

```
metrics/
├── scorers.py          # Score computation functions
├── aggregators.py      # Result aggregation (mean, std, etc.)
├── comparators.py      # Statistical comparison utilities
└── reporters.py        # Metric reporting and formatting
```

## Supported Metrics

### Classification
- Accuracy, Precision, Recall, F1-Score
- ROC-AUC, PR-AUC
- Confusion Matrix
- Matthews Correlation Coefficient

### Regression
- MAE, MSE, RMSE
- R² Score
- MAPE

## Usage

```python
from metrics.scorers import compute_metrics

scores = compute_metrics(y_true, y_pred, task='classification')
```
